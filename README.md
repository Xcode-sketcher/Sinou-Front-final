# Sinout – Frontend

## Descrição

Este repositório contém a aplicação frontend do **Sinout**, desenvolvida com **React** e **TypeScript**. O foco do projeto é oferecer uma interface de usuário moderna, responsiva e acessível, com componentes reutilizáveis e boas práticas de desenvolvimento.

## Tecnologias Utilizadas

- **React** (v19)
- **Vite** como bundler e servidor de desenvolvimento
- **ESLint** e **Prettier** para linting e formatação de código
- **Lucide** para ícones
- **Next.js** para roteamento
- **Tailwind CSS** para estilização
- **TypeScript** para tipagem estática

## Requisitos

- **Node.js** (versão 20 ou superior)
- **npm** (ou **yarn**) para gerenciamento de pacotes

## Instalação

```bash
# Clone o repositório
git clone https://github.com/Sinout-org/Sinou-Front-final.git

# Acesse o diretório do projeto
cd sinout-frontend

# Instale as dependências
npm install
```

## Executando a Aplicação

```bash
# Inicia o servidor de desenvolvimento
npm run dev
```

A aplicação estará disponível em `http://localhost:5173`.

## Scripts Disponíveis

| Script          | Descrição                                 |
|-----------------|-------------------------------------------|
| `dev`           | Inicia o servidor de desenvolvimento      |
| `build`         | Gera a versão otimizada para produção    |
| `preview`       | Visualiza a build de produção localmente   |                  |
| `lint`          | Verifica o código com ESLint              |
| `format`        | Formata o código com Prettier             |
